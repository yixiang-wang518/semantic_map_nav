### Mermaid版本

```mermaid
graph TB
    subgraph 用户层
        U1[用户终端]
        RV[RViz 可视化]
        GZ[Gazebo 仿真]
    end

    subgraph ROS中间件
        TM[ROS Master]
        TOP[话题通信<br/>/semantic_map/tags<br/>/semantic_map/markers]
        SVC[服务通信<br/>/add_tag<br/>/query_tag<br/>/delete_tag<br/>/list_tags]
        ACT[动作通信<br/>/navigate_to_tag]
    end

    subgraph 功能模块层
        TMG[语义标签<br/>管理模块]
        NAV[语义导航<br/>模块]
        MB[move_base<br/>导航核心]
    end

    subgraph 数据层
        JSON[(JSON文件<br/>持久化存储)]
        MEM[(内存<br/>标签数据)]
    end

    U1 --> TM
    RV --> TM
    GZ --> TM
    TM --> TOP
    TM --> SVC
    TM --> ACT
    TOP --> RV
    SVC --> TMG
    ACT --> NAV
    NAV --> MB
    TMG --> JSON
    TMG --> MEM
    NAV --> MEM
    MB --> NAV

    style TM fill:#f9f,color:#000
    style TMG fill:#9f9,color:#000
    style NAV fill:#ff9,color:#000
    style MB fill:#f99,color:#000
```

```mermaid
sequenceDiagram
    participant User as 用户

    User->>Nav: navigate_to_tag("meeting_room")
```

### 状态图 - 导航状态机

```mermaid
stateDiagram-v2
    [*] --> Idle: 系统启动
    Idle --> WaitingGoal: 接收Goal
    WaitingGoal --> CheckingSafety: 解析标签
    CheckingSafety --> Rejected: 危险/不存在
    CheckingSafety --> SendingGoal: 安全
    SendingGoal --> Navigating: 发送目标
    Navigating --> Succeeded: 导航完成
    Navigating --> Failed: 超时/失败
    Navigating --> Preempted: 取消请求
    Rejected --> Idle: 返回结果
    Succeeded --> Idle: 返回结果
    Failed --> Idle: 返回结果
    Preempted --> Idle: 返回结果
```
